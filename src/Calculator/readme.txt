Все команды запускать в консоли в папке проекта.

Перед использованием запустить команду:
    npm i

запуск сборки:
    npm run start
    данная команда запускает исполнение gulp task
    1) удаление папки build
    2) копирование содержимого папки public в папку build
    3) транспилирование sass, транспилирование js, копирование изображений (результаты этих действий будут помещены в папку build)
    4) запуск вотчера, который перезапускает необходимые таски в зависимости от изменяемых файлов
    5) запуск статического сервера browser-sync, который откроет в браузере по умолчанию index.html файл из папки build
    6) вотчер browser-sync, который следит за изменениями в папке build и в случае изменения файлов перезагрузит страничку в браузере

удаление папки build:
    npm run clean
